About CNC AI Academy

CNC AI Academy is an interactive, AI-powered learning platform for machinists who want to master the integration of Artificial Intelligence in modern machining. Generate custom learning modules, explore real-world projects, and advance your CNC skills with the help of top AI models.

Features:
• Module generator leveraging OpenRouter API
• Interactive, downloadable course pages
• Customizable lesson format and content

Getting Started with CNC AI Academy

Learn how to use this platform in just a few steps

Step 1: Set Your API Key

The platform uses OpenRouter to connect to AI models. Add your OpenRouter API key in the sidebar to get started.

  1. Get your API key from OpenRouter.ai
  2. Enter it in the "OpenRouter API Key" field
  3. Click "Save Key"

Step 2: Generate a Learning Module

Create custom CNC learning content tailored to your interests or challenges.

  1. Enter a topic in the "Generate New Module" section
  2. Choose your preferred learning style
  3. Click "Generate Module"

Step 3: Save Modules to Your Library

Build your personal collection of learning resources.

  • Use the "Download Current Lesson" button to save as HTML
  • Modules are automatically saved to your library
  • Favorite modules by clicking the star icon
  • Access saved modules from the "My Learning Library" section
